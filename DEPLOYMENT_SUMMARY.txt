================================================================================
                      WHAT YOU NEED TO DO
================================================================================

STEP 1: Generate JWT Keys (Do This First)
  $ openssl genrsa -out private.key 4096
  $ openssl rsa -in private.key -pubout -out public.key
  
  Then copy both keys to GitHub Secrets

STEP 2: Create Render Services
  ├─ PostgreSQL Database (grocio-postgres)
  ├─ Redis Cache (grocio-redis)
  └─ Web Service (grocio-api)

STEP 3: Add 12 GitHub Secrets
  ├─ RENDER_DEPLOY_HOOK_URL
  ├─ DATABASE_URL
  ├─ REDIS_URL
  ├─ JWT_PRIVATE_KEY
  ├─ JWT_PUBLIC_KEY
  ├─ CLOUDINARY_NAME
  ├─ CLOUDINARY_API_KEY
  ├─ CLOUDINARY_API_SECRET
  ├─ SMTP_USER
  ├─ SMTP_PASSWORD
  ├─ CORS_ORIGINS
  └─ SESSION_SECRET

STEP 4: Add 20+ Environment Variables to Render
  └─ All from .env.example

STEP 5: Push Code to Main
  $ git push origin main
  
  GitHub Actions will run automatically!

================================================================================
                      TROUBLESHOOTING
================================================================================

Common Issues:

❌ "Build fails on GitHub Actions"
   → Check: pnpm install && pnpm build --filter=@grocio/api (locally)

❌ "App crashes: DATABASE_URL is undefined"
   → Check: Render Environment tab has DATABASE_URL

❌ "Deployment doesn't trigger"
   → Check: RENDER_DEPLOY_HOOK_URL in GitHub Secrets

❌ "CORS errors from frontend"
   → Check: CORS_ORIGINS matches your frontend URL

❌ "Can't connect to database"
   → Check: DATABASE_URL is correct and PostgreSQL is running

See DEPLOY_BACKEND.md for more troubleshooting.
