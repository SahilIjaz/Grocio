API ENDPOINTS
================================================================================

1. POST /api/v1/tenants (Public)
   Register new tenant with store admin
   Status: 201 Created
   Returns: { tenant, storeAdmin }

2. GET /api/v1/tenants/:id (Auth: store_admin | super_admin)
   View tenant details
   Status: 200 OK | 404 Not Found

3. PATCH /api/v1/tenants/:id (Auth: store_admin | super_admin)
   Update tenant profile (partial)
   Status: 200 OK | 400 Bad Request

4. POST /api/v1/tenants/:id/suspend (Auth: super_admin only)
   Suspend tenant
   Status: 200 OK | 409 Conflict (already suspended)

5. POST /api/v1/tenants/:id/activate (Auth: super_admin only)
   Activate suspended tenant
   Status: 200 OK | 409 Conflict (already active)

6. GET /api/v1/tenants (Auth: super_admin only)
   List all tenants with pagination & filters
   Query: ?page=1&limit=20&status=active&search=grocery
   Status: 200 OK
   Returns: { data: [...], meta: { page, limit, total } }

7. DELETE /api/v1/tenants/:id (Auth: super_admin only)
   Delete tenant + cascade all related data
   Status: 200 OK

8. GET /api/v1/tenants/:slug/check-slug (Public)
   Check if slug is available
   Status: 200 OK
   Returns: { slug, available: true|false }
